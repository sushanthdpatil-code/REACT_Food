# 🍔 Foodie - React Food Ordering Website

A modern and responsive Food Ordering Website built using **React.js**. The application allows users to browse food items, add them to the shopping cart, manage quantities, and navigate through different pages using React Router.

---

## 📌 Features

- 🏠 Home Page
- 🍕 Menu Page
- ℹ️ About Page
- 📞 Contact Page
- 🛒 Shopping Cart
- ➕ Add to Cart
- ➖ Remove from Cart
- 🔢 Increase/Decrease Quantity
- 💰 Automatic Bill Calculation
- 🚚 Delivery Charge
- 🧾 GST Calculation
- 💳 Checkout Page
- 📱 Responsive Design
- 🎨 Modern UI with Hover Effects
- 🔄 Navigation using React Router
- ⚡ Cart Management using React Context API

---

## 🛠️ Technologies Used

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/food-ordering-react.git
```

Go to the project folder

```bash
cd food-ordering-react
```

Install dependencies

```bash
npm install
```

Install React Router

```bash
npm install react-router-dom
```

Run the project

```bash
npm start
```

The application will open at

```
http://localhost:3000
```
